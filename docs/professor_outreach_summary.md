# Professor Outreach Summary

## Short Abstract

This project develops a reproducible healthcare claims data engineering pipeline using the CMS 2008-2010 Data Entrepreneurs' Synthetic Public Use File. The system ingests raw beneficiary, inpatient, outpatient, carrier, and prescription drug event files; transforms them into analytics-ready patient-year tables; and supports utilization analysis, cost profiling, high-cost beneficiary prediction, and LLM-assisted claims explanations. Although the dataset is synthetic and not intended for real-world clinical inference, the project demonstrates scalable data modeling, longitudinal feature engineering, claims-based risk analytics, and responsible use of public healthcare data.

## Outreach Pitch

I built a public, reproducible Medicare-style claims pipeline using CMS DE-SynPUF. The project demonstrates raw claims ingestion, Bronze/Silver/Gold warehouse design, patient-year feature engineering, cost and utilization analytics, high-cost beneficiary prediction, and a safe LLM explanation layer for synthetic patient profiles.

## Skills Demonstrated

- US healthcare claims data modeling.
- DuckDB-based analytical warehouse design.
- Longitudinal patient-year feature engineering.
- Cost and utilization analytics.
- ML risk stratification workflow.
- Responsible use of synthetic public healthcare data.
- Dashboard and API delivery for applied analytics.
